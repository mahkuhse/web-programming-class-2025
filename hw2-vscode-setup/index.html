VS Code Debugger Setup

Homework 2 - How I got the debugger working

Steps:

  1. Downloaded VS Code from https://code.visualstudio.com/
  2. Installed it
  3. Opened VS Code and went to Extensions (the icon on the left)
  4. Searched for "Live Server" and installed it
  5. Made a test HTML file with some JavaScript in it
  6. Right-clicked the file and picked "Open with Live Server"
  7. It opened in my browser
  8. Pressed F12 to open the developer tools
  9. Clicked the Sources tab
  10. Clicked on the line numbers in my code to add breakpoints
  11. Reloaded the page and it stopped at the breakpoint
  12. Could see all the variable values and step through the code

What I learned:

You need Live Server to run the HTML file properly. Then you use Chrome's developer tools (F12) to debug. Click on line numbers to set breakpoints and the code will pause there so you can see what's going on.
